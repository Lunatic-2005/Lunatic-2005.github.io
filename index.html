最短路算法

1.Dijkstra算法（贪心算法）
这是一种贪心算法，其主要用到的是迭代方法。它的依据是一个重要且明显的性质：最短路是一条路，它的任意子路也是该子路两端点间的最短路。
该算法的核心思想是：从起点由近及远地求得到各点的最短路和距离，直到到达某个顶点。

下面是用python实现的算法：

  1. import heapq
  3. def dijkstra(graph, start):
  4.     # 初始化距离字典和父顶点字典
  5.     l = {vertex: float('inf') for vertex in graph}
  6.     z = {vertex: None for vertex in graph}
  7.     l[start] = 0  # 起点的距离为0
  9.     # 使用优先队列存储 (距离, 顶点) 对
  10.     priority_queue = [(0, start)]
  12.     while priority_queue:
  13.         current_distance, u = heapq.heappop(priority_queue)
  15.         # 如果当前距离大于已知最短距离，则跳过
  16.         if current_distance > l[u]:
  17.             continue
  19.         # 遍历邻接顶点
  20.         for v, weight in graph[u].items():
  21.             distance = current_distance + weight
  23.             # 如果发现更短路径，则更新
  24.             if distance < l[v]:
  25.                 l[v] = distance
  26.                 z[v] = u
  27.                 heapq.heappush(priority_queue, (distance, v))
  29.     return l, z
  31. # 示例图的邻接表表示
  32. graph = {
  33.     'A': {'B': 1, 'C': 4},
  34.     'B': {'A': 1, 'C': 2, 'D': 5},
  35.     'C': {'A': 4, 'B': 2, 'D': 1},
  36.     'D': {'B': 5, 'C': 1}
  37. }
  39. l, z = dijkstra(graph, 'A')
  40. print("最短距离:", l)
  41. print("父顶点记录:", z)
